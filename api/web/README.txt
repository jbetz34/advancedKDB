To run the web interface you must enter the file location of interface.html into a web browser.

Steps:
1.  Before sending any queries to the RDB, you must first connect to the process by clicking the connect button. 
2.  When the status bar turns green, the connection is established. You can now send queries to the RDB. 
3.  Type an instrument to filter and click "show". The first 100 records will we displayed. 

Valid Search Strings:
`BMW`GE`GW`FB`IBM.N`UL
